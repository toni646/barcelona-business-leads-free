# 📍 Base de datos GRATIS: 12.602 empresas de Barcelona (leads B2B)

Dataset gratuito de **12.602 negocios locales de Barcelona** en 30 sectores, con **teléfono normalizado, email verificado, web y dirección**. Listo para prospección B2B — descárgalo y úsalo.

👉 **[Descargar el CSV](barcelona-business-leads.csv)**

## ¿Qué incluye?

- **12.602 empresas** de Barcelona ciudad (dentistas, restaurantes, gimnasios, abogados, peluquerías, farmacias... 30 sectores).
- **~3.500 con teléfono** normalizado a formato internacional (móvil/fijo).
- **~1.600 con email verificado por DNS** (dominio que acepta correo → menos rebotes).
- Web, dirección, código postal y horario donde están disponibles.

> Calidad sobre cantidad: los emails están **verificados** (96% con dominio válido), no es una lista inflada de direcciones muertas.

## Columnas

| Campo | Descripción |
|-------|-------------|
| `sector` | Tipo de negocio |
| `name` | Nombre |
| `phone` / `phone_type` | Teléfono (E.164) y tipo móvil/fijo |
| `email` / `email_verified` | Email y si su dominio está verificado |
| `website` | Web |
| `address` / `postcode` / `city` | Dirección, CP, municipio |
| `extracted_at` | Fecha de extracción |
| `latitude` / `longitude` | Coordenadas |
